\section*{算法}
CG-CEL: 当 $E > \frac{S}{2}$ 时，先分 $\frac{c_i}{2}$，剩余用CEL分配：找 $\mu$ 使 $\sum \max(0, \frac{c_i}{2} - \mu) = E - \frac{S}{2}$

$$4(12.5-\mu)=5 \implies \mu=11.25 \implies x_i=12.5+1.25=\boxed{13.75}$$
